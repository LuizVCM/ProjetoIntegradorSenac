# 🌿TerraFar

### 📄 Observações
Este repositório tem como objetivo documentar a evolução do projeto **TerraFar**, desde a fase de design até a futura implementação.

---

##  Introdução
O **TerraFar** é um sistema que busca proporcionar maior confiabilidade na relação entre o ser humano e a natureza, atendendo à crescente demanda por segurança e informações precisas para famílias envolvidas na agricultura familiar.

Por meio da integração de **sensoriamento remoto**, **dados de satélite** e **informações climáticas em tempo real**, o TerraFar permite que agricultores tomem decisões mais assertivas sobre o manejo de suas culturas. Utilizando APIs meteorológicas, o sistema auxilia na prevenção de perdas futuras, na redução do desperdício de recursos naturais e na promoção de uma **agricultura sustentável e produtiva**.

## 🎯 Objetivo
O objetivo do TerraFar é apoiar produtores rurais oferecendo:
- Dados climáticos e ambientais confiáveis em tempo real  
- Ferramentas para monitoramento e gestão das culturas  
- Uso eficiente de insumos agrícolas  
- Suporte técnico e teórico com boas práticas de cultivo  

O sistema busca aumentar a **produtividade**, a **economia de custos** e a **sustentabilidade ambiental**.


## 👥 Requisitos do Usuário
Esta seção apresenta os requisitos funcionais do sistema TerraFar.

---

## ⚙️ Requisitos Funcionais

### 👤 Cadastro de Usuários
O sistema permite que agricultores se cadastrem e tenham acesso seguro à plataforma.  
Os dados do usuário incluem:
- ID  
- Nome completo  
- E-mail  
- Senha  
- Data de nascimento  
- Idade  
- CPF  
- Telefone  
- Saldo  

---

### 🌱 Cadastro de Sementes
Os usuários podem cadastrar sementes com as seguintes informações:
- ID  
- Tipo da semente  
- Data de plantio  

Com base nesses dados, o sistema fornece recomendações sobre:
- Tempo de germinação  
- Ambiente ideal  
- Umidade correta  
- Previsões climáticas para o plantio  

---

### 🌦️ Acesso a Dados Meteorológicos
O sistema disponibiliza dados climáticos e territoriais por meio de uma **API meteorológica**, incluindo:
- Informações climáticas locais em tempo real  
- Condições climáticas da região  
- Previsões do tempo  

---

### 🌿 Monitoramento do Crescimento das Plantas
O usuário pode acompanhar o desenvolvimento das plantas, incluindo:
- Estado atual de saúde  
- Tempo de brotamento  
- Expectativa de vida  
- Níveis de água  
- Níveis de adubos e nutrientes  

---

### 🧪 Gestão de Insumos
O sistema informa a quantidade necessária de insumos agrícolas, como:
- Fertilizantes  
- Pesticidas  
- Outros insumos essenciais  

Cada insumo é apresentado com seu nome e quantidade recomendada.

---

### 📚 Suporte Técnico e Teórico
A plataforma oferece acesso a conteúdos técnicos e educativos, incluindo:
- Melhores práticas de cultivo  
- Orientações para o cuidado com as plantas  
- Recomendações para garantir a saúde e produtividade das culturas  

---
